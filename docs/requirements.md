# Project Requirements: 

Project Idea: StudyGroup
A platform for students to create and join study groups for collaborative learning. 

Actors (3 types of users): 
• Admin 
• Group Creator 
• Student 

Requirements: 
• Different actors can log in/log out. A student can browse study groups without logging in  
but cannot join them unless logged in. 
• Admin can manage (accept or reject) newly registered accounts (Group Creators). 
• Admin can approve or reject study groups created by group creators. 
• Students can browse study groups and search based on subject, location, or meeting time. 
• Study group includes: Creator Name, Subject, Description, Maximum Members, Meeting  
Type (Online / Offline), Meeting Schedule, and Group Materials. 
• Students can request to join a study group. 
• Group Creator can accept or reject join requests. 
• Group members can share study materials and files. 
• Group members can participate in group discussions and comments. 
• Group Creator can manage groups (create, list, update, delete) (CRUD).
